Cookiebeleid Limo

Limo gebruikt cookies die het gebruik van de zoekmachine vergemakkelijken.

WAT IS EEN COOKIE?

Cookies zijn kleine tekstbestanden die je voorkeuren tijdens het surfen onthouden en opslaan op je eigen computer of toestel.

COOKIEGEBRUIK IN LIMO

Limo gebruikt enkel strikt noodzakelijke cookies.

  • Enerzijds gaat het om functionele cookies die ervoor zorgen dat bepaalde keuzes en instellingen die je een eerste keer hebt gekozen onthouden worden. Hierdoor kan je bij volgende bezoeken efficiënter, sneller en eenvoudiger zoeken in Limo. Deze cookies onthouden onder meer voorkeursinstellingen.
  • Anderzijds zijn er technische cookies die noodzakelijk zijn voor de correcte werking van de toepassing.
  • Voor gebruikers die aanmelden via het authenticatiesysteem van de eigen instelling kunnen loginnamen onthouden worden bij het aanmelden via de institutionele login.

Voor een volledig overzicht, raadpleeg onderstaande tabel:

Cookie-naam Beschrijving
_secure-[servercode] Een load balancer cookie is een klein stukje data dat door de load balancer van een website in je browser wordt opgeslagen. De load balancer is software die inkomend netwerkverkeer verdeelt over meerdere servers om te voorkomen dat een server overbelast wordt en om de prestaties van de website te verbeteren. De servercode kan variëren afhankelijk van de server waarop je sessie draait.
auth Deze cookie is nodig voor instellingen waar gebruikers via een extern authenticatiesysteem kunnen aanmelden. Zonder deze cookies kunnen we problemen ondervinden, zoals herhaaldelijk moeten aanmelden.
idpCode Deze cookies geven door welk extern authenticatiesysteem wordt gebruikt door je instelling. Ze zijn nodig voor de algemene werking van het bibliotheeksysteem. De code verwijst naar het authenticatiesysteem van je instelling.
institute Deze cookies geven de code van je instelling weer. Ze zijn nodig voor de algemene werking van het bibliotheeksysteem.
JSESSIONID Deze cookie slaat de code van de Limo-sessie op. Dit wordt gebruikt door het bibliotheeksysteem om het aantal sessies te monitoren.
productCode Deze cookies zijn nodig voor de algemene werking van het bibliotheeksysteem. De code verwijst naar het type bibliotheeksysteem dat wordt gebruikt.
urm_se Deze cookie helpt de sessie te beheren en te onderhouden terwijl je het bibliotheeksysteem gebruikt. Zonder deze cookies kunnen we problemen ondervinden, zoals verlies van sessiegegevens.

WIJZIGEN VAN BROWSERINSTELLINGEN

Cookies kan je via de browserinstellingen op je computer weigeren of verwijderen. De manier waarop je dit kan doen, verschilt van browser tot browser. Raadpleeg indien nodig de helpfunctie van je browser.
Noot: Deze instellingen moet je steeds per browser en per computer apart instellen!

Het niet-aanvaarden, verwijderen of uitschakelen van cookies kan tot gevolg hebben dat bepaalde functionaliteiten van Limo niet meer toegankelijk zijn of slecht functioneren. Daardoor kan de beleving van de website tegenvallen.
