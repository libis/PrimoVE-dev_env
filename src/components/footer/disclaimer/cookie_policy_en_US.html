LIMO COOKIE POLICY

Limo uses cookies that facilitate the use of the search engine.

WHAT IS A COOKIE?

Cookies are small text files that remember your preferences while browsing and store them on your own computer or device.

COOKIE USE IN LIMO

Limo uses only strictly necessary cookies.

  • On the one hand, functional cookies are used to ensure that certain choices and settings you have selected on your first visit are remembered. This allows you to search Limo more efficiently, quickly and easily on subsequent visits. Among other things, these cookies remember preference settings.
  • On the other hand, technical cookies are necessary for the application to function properly.
  • For users logging in via their own institution's authentication system, login names may be remembered when logging in via the institutional login.

For a complete overview, see to the table below:

Cookie Name Description
_secure-[server code] A load balancer cookie is a small piece of data stored in your browser by a website's load balancer. The load balancer is a piece of software that distributes incoming network traffic across multiple servers to prevent server overload and to improve website performance. The server code may vary depending on the server that is handling your session.
auth This cookie is required for institutions that allow users to log in via an external authentication system. Without these cookies, Limo may experience problems, such as having to log in repeatedly.
idpCode These cookies tell Limo which external authentication system is being used by your institution. They are necessary for the general working of the library system. The code refers to your institution's authentication system.
institute These cookies display the code of your institution. They are necessary for the general working of the library system.
JSESSIONID This cookie stores the Limo session code. This is used by the library system to monitor the number of sessions.
productCode These cookies are necessary for the general working of the library system. The code refers to the type of library system used.
urm_se This cookie helps manage and maintain the session while you use the library system. Without these cookies, Limo may experience problems, such as loss of session data.

CHANGING BROWSER SETTINGS

You can refuse or delete cookies via the browser settings on your computer. How you can do this varies from one browser to another. Consult the help function of your browser if necessary.
Note: These settings must be set separately for each browser and computer!

  • Cookie settings in Edge
  • Cookie settings in Chrome
  • Cookie settings in Firefox
  • Cookie settings in Safari

Refusing to accept, deleting or disabling cookies may result in certain features of Limo becoming inaccessible or working with reduced functionality. As a result, the user experience of the website may be negatively affected.
