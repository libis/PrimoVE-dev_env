Wat is de Limo AI-assistent?

AI komt naar Limo! Met de Limo AI-Assistent kan je je zoekvraag in natuurlijke taal stellen. Met behulp van GenAI zet de tool deze om naar een zoekopdracht die de zoekmachine begrijpt:

Klik op de 'Limo AI-assistent'-knop in het menu om met deze tool aan de slag te gaan:

  • Krijg een eerste indruk van je zoektopic met een kort antwoord op basis van GenAI, voorzien van inline citaties
  • Verken meer bronnen via "Bekijk meer resultaten"
  • Doe inspiratie op voor aanverwante zoektopics dankzij de "Gerelateerde zoekvragen" onder het overzicht

Hoe een zoekvraag stellen in de Limo AI-assistent?

  • Stel je vraag zoals je die aan een persoon zou stellen, met een vraagteken
  • Wees zo concreet en specifiek mogelijk, gebruik vaktermen
  • Je kan vragen in het Nederlands stellen, ze worden automatisch vertaald​
  • Je kan beperken op type document en op recente publicaties
